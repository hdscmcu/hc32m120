功能描述
================================================================================
本样例主要展示TIMER0的捕获功能，在捕获中断中将捕获的计数值读出并保存。

说明：
使能TIMER0 输入捕获中断功能，异步计数方式，LRC作为计数时钟源频率32.768KHZ。
同时使能INTC1（SW2）下降沿和上升沿产生内部触发事件，通过事件触发TIMER0输入捕获功
能，进入TIMER0中断。

无

================================================================================
使用步骤
================================================================================
1） 打开工程编译并全速运行。
2） 短按按键SW2，使产生INTP1的下降沿和上升沿。
3） 绿灯常亮，表示进入两次输入捕获中断，并且采集捕获值完毕。

================================================================================
注意
================================================================================
同步计数方式下时钟源可选HCLK、内部硬件触发事件
异步计数方式下时钟源只能是LRC
