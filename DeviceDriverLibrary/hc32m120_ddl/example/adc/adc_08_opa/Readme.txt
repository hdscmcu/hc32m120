功能描述
================================================================================
本例程展示了OPA模块的基本用法。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
STK_HC32M120_LQFP48_050_V11

辅助工具:
---------------------
USB转串口工具

辅助软件:
---------------------
串口调试助手

================================================================================
使用步骤
================================================================================
1）测试时，可分别为OPA1和OPA2设置不同的放大倍数，并按用户手册上OPA的外围电路和电气特性
   选择合适的Rn和Rp；
2）为测试OPA的工作情况，可将电位器的输出与OPA1和OPA2的输入短接，通过电位器来提供不同的电压；
3）USB转串口工具连接至USART3，并与电脑相连；用MicroUSB将开发板与电脑相连；
4）将电位器旋转至输出电压为0的位置；
5）打开工程，重新编译，下载至开发板并运行；
6）根据放大倍数，将电位器在合理的范围内旋转，可通过串口调试助手看到放大前和放大后的电压；
7）也可用断点调试的方式来观察程序运行情况。


================================================================================
注意
================================================================================
请严格按照OPA之电气特性使用OPA。
