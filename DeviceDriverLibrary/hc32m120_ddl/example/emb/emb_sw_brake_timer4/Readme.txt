功能描述
================================================================================
本样例主要展示EMB模块的的软件触发制动信号功能。

说明：
无

================================================================================
测试环境
================================================================================
测试用板:
---------------------
STK_HC32M120_LQFP48_050_V11

辅助工具:
---------------------
示波器Tektronix DPO2024B

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）测试板J3-P61、J3-P60引脚与示波器连接
2）编译、下载并运行
3）观察示波器，示波器J3-P61、J3-P60通道显示周期脉冲（1Hz）
4）按下并释放SW2按键
5）观察示波器，J3-P61、J3-P60通道电平为低
6）按下并释放SW2按键
7）观察示波器，示波器J3-P61、J3-P60通道显示周期脉冲（1Hz）
8）重复步骤3） ~步骤7）
