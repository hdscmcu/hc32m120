功能描述
================================================================================
本样例为EVENT_PORT样例。

说明：
本样例通过使用MCU的IO口产生EVENT_PORT的触发信号，并在触发时获取当时EVENT_PORT管
脚状态来展示EVENT_PORT功能。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
STK_HC32M120_LQFP48_050_V11

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）使用3个跳线帽，分别短接UART1-CK&CTSRTS, UART2-CK&CTSRTS, UART3-CK&CTSRTS；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能，全速运行；
4）LED_G会在EVENT_PORT中断服务程序中交替亮灭
