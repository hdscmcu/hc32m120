功能描述
================================================================================
本样例主要展示如何使用Timer4比较输出模块OCO双通道。

说明：
样例配置比较输出模块OCO双通道和CNT计数值匹配时，产生中断信号，中断服务程序回调函
数翻转I/O。

测试环境
================================================================================
测试用板:
---------------------
STK_HC32M120_LQFP48_050_V11

辅助工具:
---------------------
示波器Tektronix DPO2024B

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）测试板J3-P63 IO引脚与示波器连接；
2）编译、下载并运行；
3）观察示波器，示波器显示周期脉冲（1Hz），占空比50%，周期时间与CNT周期一致。
