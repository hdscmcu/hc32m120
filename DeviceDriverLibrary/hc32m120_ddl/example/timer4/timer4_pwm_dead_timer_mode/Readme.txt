平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示如何使用Timer4PWM死区定时器模式。

说明：
样例配置比较输出模块OCO低通道，并通过PWM死区定时器模式，输出其波形

使用步骤
================================================================================
1）测试板J3-P61(TIM4_1_OUH)、J3-P60(TIM4_1_OUL)引脚与示波器相连；
2）编译、下载并运行；
3）观察示波器，J3-P61、J3-P60通道波形周期相同，且周期时间为Timer4 Counter计数器
   周期2倍。TIM4_1_OUH占空比为1/2，TIM4_1_OUL占空比为1/4
